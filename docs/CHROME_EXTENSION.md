# Как поставить расширение в Chrome

## Откуда грузить

Папка **`extension/`** в этом репозитории — это и есть расширение. Сборка не нужна.

На твоём Mac путь:

```text
/Users/deniscivkunov/Desktop/Плагин для автодокументации/extension
```

Если клонировал с GitHub — выбери `…/training-recorder/extension`.

## Установка (1–2 минуты)

1. Chrome → адресная строка: `chrome://extensions/`
2. Справа вверху: **Режим разработчика** — включить
3. **Загрузить распакованное расширение**
4. Выбрать папку **`extension`** (см. путь выше)
5. Иконка **Training Recorder** появится на панели

## Первый запуск

1. Нажми иконку расширения
2. Поле **«Адрес сервера»** — уже стоит прод:

   `https://training-recorder-production.up.railway.app`

   (для локальной отладки можно `http://127.0.0.1:8000`)
3. Поставь галочку согласия → **Запись**
4. Разреши захват вкладки и микрофон
5. После **Стоп** → **Отправить и открыть редактор**

## Проверка, что сервер жив

```bash
curl -sS https://training-recorder-production.up.railway.app/health
```

Ожидается: `{"status":"ok"}`

## Обновление расширения после `git pull`

Chrome → `chrome://extensions/` → у Training Recorder кнопка **Обновить** (↻).
